# ⚽ Moxie-Gotchi Arena v5.0 - Edición Pasión y Rivalidad Mundialista

Este proyecto fue construido inspirándose en la **pasión desbordante del fútbol, el espíritu de equipo y la eterna rivalidad de los derbis mundialistas**. Es un simulador y arena estratégica web donde gestionas la energía, tácticas e hinchada de tu club mientras compites contra rivales clásicos. 

El sistema implementa una arquitectura desacoplada cliente-servidor enfocada en la robustez lógica y seguridad en el backend.
